# Embodied-Intelligence-Stack-EIS-
A three‑stage exploration of embodied intelligence. EIS builds from First-Principles RL to robotics and perception, and Bayesian active inference. Designed for reproducible research in learning, action, and inference.

## FROM RL to Active Inference (Bayesian Inference)
- L1 focuses on exploring Reinforcement Learning from First-Principles.
- L2 integrate Deep RL with an agent that uses SLAM and perception (LiDAR)
- L3 Investigates Bayesian Inference by integrating active inference with Reinforcement Learning on an agent (robotics and perception from Level-2)
